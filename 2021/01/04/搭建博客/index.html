复制其代码将它放在_config.yml里，下载（博客主目录，git bash here），再将use修改为自己选择的模型的代码，然后在gitbush输入那三条命令，登录网址可以看到动画模型。

做完所有效果之后，打开github创建自己的账户，QQ邮箱验证，在创建自己的数据库，网址名为：用户名+github+io（中间都有.），然后在blog里打开自己gitbush里输入（因为做的时候忘截图，就将代码打至如下）：

git config –global user.name “2065970559” （输入自己github上的用户名）

git config –global user.email “2065970559@qq.com” （输入自己注册时候的邮箱）

ssh-keygen -t rsa -C “2065970559@qq.com” （这条命令使我们电脑上出现ssh的文档为了形成公钥，从而和github形成连接）

再回车，输入两遍自己github的密码就好了。
